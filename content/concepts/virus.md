# Virus

:::definition
וירוס (Virus) הוא נוזקה שנצמדת לקובץ מארח (host file) — מסמך או קובץ הרצה — ואינה
יכולה לרוץ בלעדיו. הרצת המארח מריצה גם את קוד הווירוס, ואז הוא משכפל את עצמו
ומתפשט. הפעלת המארח דורשת אינטראקציה של המשתמש.
:::

## הסבר פשוט

וירוס אינו תוכנית עצמאית. הוא **טפיל**: כדי להתקיים הוא חייב להיכנס לתוך קובץ של
מישהו אחר — מסמך Word, קובץ EXE, תוכנה מוכרת ולגיטימית. כל עוד אף אחד לא פותח את
הקובץ הזה, הווירוס שוכב שם ולא עושה דבר.

מכאן נובע המאפיין השני, וזה שקובע: הווירוס **זקוק לכם**. מישהו צריך להוריד את
הקובץ, ללחוץ עליו, לפתוח אותו, ולעיתים גם לאשר הפעלת **Macros**. רק אז קוד
הווירוס רץ, משכפל את עצמו לקבצים נוספים, ומבצע את ה-**Payload** שלו — גניבת מידע,
השחתה או המשך הפצה.

זו הנקודה שהמרצה מדגיש כ-Key Insight מפורש:
**"A virus always needs a host file and user action to activate."**

## הסבר טכני

הצמדת הווירוס למארח מוצגת בשתי רמות. **פיזית** — קוד הווירוס מוזרק לתוך הקובץ
הקיים. **לוגית** — הקובץ הנגוע נראה כרצף של שלושה חלקים: Virus Code Part (a) →
Original Program → Virus Code Part (b). התוצאה: המשתמש מריץ את מה שהוא חושב
שהוא התוכנית המקורית, והתוכנית אכן רצה — אבל קוד הווירוס רץ יחד איתה.

המרצה מבחין בין שני סוגי וירוסים לפי סוג המארח:

- **Document-Based Virus** (למשל ‎.DOC‎) — הווירוס יושב ב-**Macros** של המסמך. יש
  כאן שלב הפעלה **נוסף**: המסמך מבקש מהמשתמש `enable macros`, ורק האישור הזה
  מפעיל את הקוד.
- **Executable-Based Virus** (למשל ‎.EXE‎) — הווירוס מופעל **מיידית** עם הרצת
  הקובץ, משתלט על משאבי המערכת כדי לשכפל את עצמו, ומתפשט לקבצים נוספים ולהתקנים
  מחוברים.

וירוס מתקדם משתמש ב-**Polymorphism**: בכל שכפול הוא משנה את הקוד הבינארי של עצמו,
כך שלכל עותק יש חתימה (signature) אחרת. התוצאה היא שאין חתימה יציבה להתאים אליה,
ולכן [[Knowledge-based Detection]] — שמבוסס כולו על השוואה לחתימות ידועות —
מפספס אותו.

:::example
עובדת בחברת ביטוח מקבלת בדוא"ל קובץ Word בשם "חשבונית_2025.doc". היא פותחת אותו,
והמסמך מציג פס צהוב: "Enable Content". היא מאשרת. ה-**Macros** רצות, קוד הווירוס
מופעל, והוא מתחיל לשכפל את עצמו למסמכים אחרים בתיקייה המשותפת. Microsoft Word הוא
ה-**Host Program** — בלעדיו לא היה קוד שירוץ.
:::

:::warning
וירוס אינו מתפשט לבדו ברשת. [[Worm]] הוא זה שמתפשט אוטומטית בין מחשבים בלי
מארח ובלי משתמש; וירוס נעצר עד שמישהו יפתח את הקובץ. סטודנטים שמצמידים לווירוס
את התכונה "התפשטות עצמאית" עונים בעצם על תולעת.
:::

:::diagram
תרשים בשני חצאים. משמאל, "Physically": קופסה בשם Virus Code עם חצים שנכנסים לתוך
קופסה גדולה בשם Original Program — קוד הווירוס נדבק פיזית לקובץ הקיים. מימין,
"Logically": עמודה בת שלושה בלוקים זה מעל זה — Virus Code Part (a), Original
Program, Virus Code Part (b) — שמראה איך הקובץ הנגוע נראה מבחוץ. מתחת לשני החצאים
כיתוב אחד: הרצת המארח = הרצת הווירוס.
:::
