# UI (User Interface)

:::definition
ממשק משתמש (UI - User Interface) הוא נקודת המגע הפיזית או הדיגיטלית שדרכה מתבצעת התקשורת והעברת המידע בין המשתמש האנושי לבין המערכת הטכנולוגית.
:::

## הסבר פשוט

במילים פשוטות, UI הוא כל מה שאתם רואים, שומעים או נוגעים בו כשאתם מפעילים מכשיר טכנולוגי. זה כולל את הכפתורים שמופיעים על מסך הטלפון, שדות הטקסט שבהם אתם מקלידים, הצבעים של האפליקציה, האייקונים, הגופנים, ואפילו רכיבים פיזיים כמו הגה של מכונית או השלט של המזגן. ה-UI הוא הצינור המתווך שמנגיש לכם את הקוד הפנימי של המערכת בצורה מוחשית.

ללא ממשק משתמש, כדי להפעיל תוכנות היינו צריכים לכתוב פקודות קוד ישירות לחומרה של המחשב.

## הסבר טכני

ממשק המשתמש מתוכנן כדי לאפשר למשתמש לבצע פעולות (Input) ולקבל משוב על תוצאותיהן (Output). קיימים מספר סוגי ממשקים עיקריים שהתפתחו לאורך ההיסטוריה של המחשוב:

1. **ממשק משתמש גרפי (GUI - Graphical User Interface)** — הממשק המודרני הנפוץ ביותר, המבוסס על ייצוגים ויזואליים של מידע (כמו חלונות, כפתורים ותפריטים) שנשלטים על ידי עכבר, מקלדת או מגע.
2. **ממשק שורת פקודה (CLI - Command Line Interface)** — ממשק טקסטואלי בלבד, שבו המשתמש מזין פקודות מוגדרות מראש דרך המקלדת ומקבל פלט טקסטואלי חזרה. ממשק זה מהיר וחסכוני במשאבים, אך דורש עקומת למידה גבוהה.
3. **ממשק קולי (VUI - Voice User Interface)** — ממשק המאפשר אינטראקציה באמצעות דיבור חופשי או פקודות קוליות, תוך שימוש במערכות זיהוי דיבור ועיבוד שפה טבעית (NLP).
4. **ממשק פיזי/מדיום קשיח (Tangible UI)** — כפתורים פיזיים, מסכי מגע, גלגלות ורכיבים מכניים המצויים במכשירים יומיומיים או תעשייתיים (כמו דשבורד של רכב או לוח בקרה של מכונת כביסה).

:::example
באפליקציית השמעת מוזיקה כמו Spotify, ה-UI כולל את כפתור ה-Play הירוק, רשימת השירים, בר הניווט התחתון, שורת החיפוש, הצבעים הכהים של הרקע ותמונות האלבומים שמוצגות למשתמש.
:::

:::warning
UI אינו UX. ממשק המשתמש (UI) עוסק בהיבטים החזותיים והטכנולוגיים של הממשק – איך הוא נראה ומעוצב ברמת המסך. חוויית המשתמש (UX) רחבה בהרבה ועוסקת בתחושה הכללית של המשתמש, קלות השימוש ויעילות ההגעה למטרה שלו במערכת כולה.
:::
