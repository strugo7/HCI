# Command and Control

:::definition
Command and Control (C&C / C2 — שרת שליטה ובקרה) הוא שרת בשליטת התוקף, ש[[Malware]]
שכבר רצה על מחשב הקורבן מתקשרת אליו כדי לקבל הוראות, מפתחות או payload נוסף, וכדי
לדווח לתוקף ("phone home"). ערוץ ה-C&C הוא מה שהופך הדבקה חד-פעמית לשליטה מרחוק
מתמשכת.
:::

## הסבר פשוט

נוזקה שרצה על המחשב שלכם היא, בפני עצמה, חתיכת קוד שקטה. מה שנותן לה חיים הוא הקשר
החוזר אל התוקף — ערוץ שדרכו היא שואלת "מה לעשות עכשיו?" ומקבלת תשובה. השרת שבצד
השני של הקשר הזה נקרא שרת שליטה ובקרה (Command and Control).

דרך הערוץ הזה התוקף יכול לשלוח פקודות (גנוב קובץ, הקלט הקשות, התפשט הלאה), להוריד
כלים נוספים, או — במקרה של כופרה — למסור את מפתח ההצפנה. בלי הערוץ הזה, הרבה
מתקפות פשוט נעצרות.

## הסבר טכני

המרצה מלמד את מנגנון ה-C&C במלואו דווקא בהקשר ה-Ransomware וה-Dropper:

- **בכופרה** (שקופיות 18–19): הנוזקה "מקבלת מפתח מ-C&C server" ו-"downloads public
  key" דרך **TOR**. כלומר ה-C&C הוא המקור של ה-Public key שבלעדיו אין הצפנה.
- **בדרופר** (שקופית 22): השלב החמישי בשרשרת הוא **C2 Beaconing** —
  "Malware phones home to the attacker's command-and-control server". רק אחרי
  ה-beaconing מגיע ה-Remote Control, שבו התוקף מוציא פקודות בפועל.

מבחינת הגנה, ערוץ ה-C&C הוא נקודת תורפה של התוקף — הוא מייצר תעבורה יוצאת שאפשר
לזהות ולחסום. במצגת ה-NGFW/IDS (06, שקופית 43) זהו בדיוק מה ש-[[NGFW]] חוסמת ומה
שחומת אש ישנה (מבוססת פורטים בלבד) מפספסת, מפני שתעבורת C&C מודרנית מתחבאת בתוך
פרוטוקולים מותרים כמו HTTPS.

:::example
בדיאגרמת ה-Gh0st RAT (שקופית 23): Spear Phishing → הקורבן לוחץ → [[Dropper]] יורד
→ הדרופר מוריד את Gh0st RAT → **C2 Beaconing (Gh0st Protocol)** → התוקף מוציא פקודה.
דוגמאות לפקודות שהמרצה מונה: צילום ה-webcam, הורדת/העלאת קבצים, keystroke logging,
remote shell, ו-system inventory. כל אלה עוברים דרך ערוץ ה-C&C.
:::

:::warning
**C&C אינו שלב ההדבקה — הוא מה שבא אחריו.** ההדבקה (phishing, dropper) רק מכניסה את
הנוזקה פנימה. ערוץ ה-C&C הוא מה שממנו התוקף שולט. לכן בשאלת ה-Dropper במבחן (2023,
שאלה 4) התשובה הנכונה מדגישה שהדרופר "יוצר שליטה וקשר עם התוקף (C&C)" — הדרופר עצמו
אינו מזיק, הוא מקים את הערוץ.
:::

:::
