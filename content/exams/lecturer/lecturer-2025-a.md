# מבחן מרצה — HCI, תשפ"ה, מועד א'

Human-Computer Interface, 273020 · ד"ר משה ליבה · 26/02/2025 · 27 שאלות · משקל זהה · ללא חומר עזר.
השאלות והמסיחים הועתקו מילה במילה מהטופס המקורי (באנגלית). ההסברים, מטרות הלמידה ותשומת הלב למסיחים נכתבו על ידינו — המרצה לא פרסם מפתח תשובות או הסברים.

## Question

id: q-lecturer-2025-a-001
type: multiple-choice
difficulty: easy
cognitive: understand
points: 4
concepts:
  - ux

When designing a voice-based virtual assistant, the biggest UX challenge is often:

### Answers

A. Maintaining consistent color palettes.

B. Using traditional mouse-click interactions.

C. Handling natural language inputs and providing clear spoken feedback.

D. Keeping the code size minimal.

E. Avoiding brand references in marketing copy.

Correct: C
Explanation: בממשק קולי אין מסך עשיר לרמזים חזותיים, ולכן האתגר המרכזי הוא הבנת קלט טבעי ומגוון של הדובר ומתן משוב קולי ברור על מצב המערכת. פלטת צבעים, אינטראקציית עכבר וגודל קוד אינם רלוונטיים למודאליות קולית, ושיקולי מיתוג שיווקי אינם נוגעים לאתגר החוויה.
Learning Objective: לזהות שהאתגר המרכזי בממשק קולי הוא עיבוד שפה טבעית ומשוב ברור, ולא היבטים ויזואליים.
Misconception: סטודנטים נוטים להעתיק שיקולי עיצוב מסכי (צבע, כפתורים) לממשק קולי שבו הם אינם קיימים.

In a heuristic evaluation, experts typically:

### Answers

A. Conduct large-scale user surveys.

B. Check the interface against known usability best practices (e.g., Nielsen's).

C. Fix code-level errors in real time.

D. Focus only on visual design consistency.

E. Ignore minor usability issues entirely.

Correct: B
Explanation: הערכה היוריסטית היא בדיקה של מומחים המשווים את הממשק לכללי אצבע מוכרים של שמישות (כמו העשרת ההיוריסטיקות של נילסן). היא אינה סקר משתמשים בקנה מידה גדול, אינה תיקון קוד בזמן אמת, אינה מוגבלת לעקביות חזותית, ובוודאי אינה מתעלמת מבעיות קטנות — אלה דווקא מתועדות.
Learning Objective: להגדיר הערכה היוריסטית כבדיקת מומחים מול עקרונות שמישות ידועים.
Misconception: בלבול בין הערכת מומחים (ללא משתמשים) לבין מחקר משתמשים בקנה מידה גדול.

### Answers

Now consider a 3D racing video game interface. Which aspects of the five usability factors are typically most important?

A. A balanced mix of Learnability, Memorability, Errors, Satisfaction, and Efficiency.
