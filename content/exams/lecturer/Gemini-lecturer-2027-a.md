Which modification to the kiosk interface best applies Norman's concept of signifiers to resolve user confusion?

### Scenario

Users of a public transit kiosk frequently tap a non-interactive static banner displaying ticket prices, assuming it is a button, while failing to notice the actual touchable "Buy Ticket" area located at the bottom corner.

### Answers

A. Replace the static text banner with a glowing border and directional icon while making the actual button visually prominent.

B. Remove all price information from the initial screen to prevent users from looking at the banner.

Which design change to the telemetry interface most effectively reduces extraneous cognitive load based on Cognitive Load Theory?

### Scenario

An industrial monitoring console displays live pressure graphs on screen 1, operating threshold tables on screen 2, and warning alert pop-ups on screen 3. Operators during high-stress incidents report severe mental fatigue and delayed response times as they attempt to integrate information across all three screens.

### Answers

A. Embed relevant operating thresholds directly onto the live pressure graphs alongside inline alert badges.

B. Display all numeric raw data on screen 1 while moving all visual graphs exclusively to screen 3.

C. Require operators to memorize operating threshold values during onboarding to clear screen 2 entirely.

D. Embed relevant operating thresholds directly onto a separate modal window that opens over screen 2.

E. Introduce animated visual transitions between screens to draw the operator's focus during emergencies.

Correct: A
Explanation: שילוב ערכי הסף הטלמטריים ישירות על גבי הגרפים המצוירים (Inline) מונע את אפקט פיצול הקשב (Split-attention effect) ומפחית את העומס הקוגניטיבי החיצוני (Extraneous cognitive load).
Learning Objective: ליישם עקרונות מתאוריית העומס הקוגניטיבי לצמצום פיצול קשב בממשקים מרובי מסכים.
Misconception: הנחה שהפרדת מידע למסכים נפרדים מקלה על המפעיל במקום לייצר עומס קוגניטיבי מיותר.
