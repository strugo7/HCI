Which of the following statements best explains why recognition is cognitively easier for humans than recall?

### Answers

A. Recognition relies on external visual or auditory cues that directly trigger associated memory chunks, whereas recall requires scanning memory unaided.

B. Recognition operates entirely within sensory registers and bypasses short-term storage, whereas recall requires activation of long-term memory.

C. Recognition requires the user to memorize exact commands and keywords, whereas recall uses flexible visual icons to represent system functionalities.

D. Recognition is only accessible to users with advanced technical training, whereas recall is a natural skill developed during early childhood.

Correct: A
Explanation: זיהוי קל יותר מפני שהגירוי החיצוני המוצג (כמו מילה, כפתור או אייקון) משמש כרמז שליפה (Retrieval Cue) המפעיל ישירות את המידע השמור בזיכרון. בהיזכרות, המשתמש חייב לסרוק את הזיכרון בעצמו ללא כל סיוע חיצוני. התשובות האחרות מעוותות את הגדרת הזיכרון או הופכות את היחס בין הכלים.
Learning Objective: Explain the cognitive difference between recognition and recall.
Misconception: Students often swap the roles of recognition and recall or believe recognition doesn't involve long-term memory.

Which interface paradigm shift would best address this cognitive load by prioritizing recognition?

### Scenario

A system administrator complains that a newly deployed database management terminal requires them to memorize over thirty syntax commands to perform daily tasks, leading to frequent syntax errors.

### Answers

A. Transitioning from a Command Line Interface (CLI) to a Graphical User Interface (GUI) that displays commands as selectable menu actions.

B. Increasing the terminal font size and adding color coding to highlight syntax errors in real-time as the administrator types them.

C. Dividing the terminal interface into multiple smaller windows to increase the working memory capacity of the administrator during tasks.

D. Requiring the administrator to undergo a intensive training program to memorize all thirty syntax commands before using the system.

Correct: A
Explanation: ממשק שורת פקודה (CLI) מתבסס לחלוטין על היזכרות (Recall) ומאלץ את המשתמש לזכור פקודות בעל-פה. מעבר לממשק גרפי (GUI) המציג את הפעולות האפשריות כפתורים או תפריטים נפתחים הופך את המשימה לזיהוי (Recognition) ומפחית משמעותית את העומס הקוגניטיבי ואת טעויות ההקלדה. פתרונות אחרים רק משפרים קריאות או מנסים לאלץ למידה מבלי לשנות את אופי משימת הזיכרון.
Learning Objective: Distinguish between CLI and GUI in terms of memory demands.
Misconception: Students sometimes focus on visual design cosmetics (like font sizes or colors) rather than the underlying memory retrieval mechanism.

## Question

difficulty: hard
cognitive: apply

concepts:

Which feature would best apply the 'Recognition over Recall' principle to solve this?

### Scenario

An e-commerce mobile application requires users to type the exact brand name of products they purchased in the past to find them for reordering. User testing shows high cart abandonment.

### Answers

A. Displaying a 'Buy It Again' section showing photos and names of previously purchased items on the home screen.

B. Adding a detailed search guidelines page explaining how to format and spell brand names in the search input.

C. Enlarging the search input box and placing it at the very top of the homepage to maximize its visual prominence.

D. Sending daily promotional emails listing the top twenty most popular products across all categories of the store.

Correct: A
Explanation: הצגת סעיף 'קנה שוב' עם תמונות ושמות של מוצרים שנרכשו בעבר מאפשרת למשתמש לזהות אותם חזותית ובקלות (Recognition) ללא צורך להיזכר בשמות המותגים המדויקים ולהקלידם (Recall). עמודי הנחיות חיפוש או הגדלת שדה הקלט עדיין מאלצים את המשתמש להיזכר, ומיילים פרסומיים אינם פותרים את קושי האיתור בממשק.
Learning Objective: Apply 'Recognition over Recall' design patterns to improve usability.
Misconception: Students often propose solutions that ease typing or search entry (like guidelines) rather than eliminating the memory retrieval task entirely.
