## Question

id: core-visual-principles/q1
type: multiple-choice
difficulty: easy
cognitive: remember
estimatedTime: 45
points: 5
concepts:
  - design-principles

Which of the following is NOT one of the five basic **elements** of design?

### Answers

A. Color

B. Emphasis

C. Texture

D. Shape

Correct: B
Explanation: Emphasis הוא אחד מתשעת **עקרונות** העיצוב, לא אחד מאלמנטי הבסיס. חמשת האלמנטים הבסיסיים של עיצוב הם: Color (צבע), Shape (צורה), Texture (מרקם), Space (מרחב) ו-Form (צורה תלת-ממדית). כל האחרים — Balance, Emphasis, Movement, Pattern, Repetition, Proportion, Rhythm, Variety ו-Unity — הם עקרונות.
Learning Objective: List the five basic design elements and the nine design principles.
Misconception: Students often confuse design elements (the raw materials) with design principles (the rules for using them). Emphasis is a key principle, not a raw element.

## Question

id: core-visual-principles/q2
type: scenario
difficulty: medium
cognitive: analyze
estimatedTime: 60
points: 5
concepts:
  - design-principles

Which design problem best describes the interface in the following scenario?

### Scenario

A user opens a dashboard app for tracking fitness goals. On a single screen they see: 7 different bright colors, 4 charts of equal size with equal font weights, 12 action buttons all styled identically, and text in 6 different fonts. The user stares at the screen for 10 seconds and does not know where to look or what to do first.

### Answers

A. The interface is unbalanced — visual weight is concentrated on one side of the screen.

B. The interface is unfocused — there is no clear visual hierarchy or emphasis point to guide the user.

C. The interface suffers from poor color theory — the palette choices are technically incorrect.

D. The interface is well-designed; the user simply lacks experience with fitness apps.

Correct: B
Explanation: הממשק המתואר סובל מהיעדר דגש (Emphasis) ופרופורציה (Proportion). כאשר 4 תרשימים בגודל שווה, 12 כפתורים זהים ו-6 גופנים שונים מתחרים על תשומת הלב — לא קיים מוקד ויזואלי ראשי. המשתמש לא יכול לדעת מהי הפעולה החשובה ביותר. זהו כשל של חוסר מיקוד (Unfocused), לא חוסר איזון (שהוא כשל שונה לגמרי).
Learning Objective: Identify the three main visual design failures: unbalanced, cluttered, and unfocused.
Misconception: Students often confuse "cluttered" (too many competing elements) with "unfocused" (no clear emphasis). In this scenario the problem is specifically the absence of a clear focal point, not just visual overload.

## Question

id: core-visual-principles/q3
type: multiple-choice
difficulty: medium
cognitive: apply
estimatedTime: 60
points: 5
concepts:
  - design-principles

A product designer is building an e-commerce product card. They make the product image large, the product name medium-sized, the price in a contrasting accent color but small, and the description in a tiny gray font. Which design principle is the designer primarily applying?

### Answers

A. Rhythm — creating a visual beat through repeating elements.

B. Unity — ensuring all elements belong to the same visual family.

C. Proportion — using relative size to communicate importance hierarchy.

D. Pattern — repeating a fixed visual structure across all cards.

Correct: C
Explanation: המעצב משתמש בעיקרון ה**פרופורציה (Proportion)** — יצירת יחסי גודל שמשקפים את חשיבות האלמנטים. התמונה הגדולה ביותר (הפריט החשוב לרכישה), שם המוצר בינוני, המחיר בצבע בולט (חשוב אך קטן יותר), והתיאור קטן ביותר. הסדר הזה מוביל את עין הקורא בצורה מכוונת. Rhythm קשור לתדירות/ריווח, Unity לשפה ויזואלית אחידה, ו-Pattern לחזרה של מבנה קבוע.
Learning Objective: Apply the principle of Proportion to create a clear visual hierarchy in interface elements.
Misconception: Students sometimes confuse Proportion with Pattern. Pattern is about repeating the same structure across multiple items; Proportion is about size relationships within a single item to establish hierarchy.

## Question

id: core-visual-principles/q4
type: multiple-choice
difficulty: hard
cognitive: evaluate
estimatedTime: 75
points: 5
concepts:
  - design-principles

A designer uses the exact same button style, color scheme, and font across every screen of an app. A colleague argues this is "boring" and suggests adding random visual variations on each screen. Which response best defends the designer's approach?

### Answers

A. The colleague is correct — Variety is a design principle and every screen should look different to maintain user interest.

B. The designer is applying Repetition intentionally: consistent visual elements build a shared visual language the user learns once, reducing cognitive load on every subsequent screen.

C. Both are wrong — the best solution is to use Repetition on some screens and Variety on others, randomly selected.

D. The designer is applying Pattern, which only applies to lists and tables, not to overall screen design.

Correct: B
Explanation: המעצב מיישם בצורה מכוונת את עיקרון **החזרתיות (Repetition)** — עקביות ויזואלית בין מסכים. כאשר המשתמש לומד את "השפה הויזואלית" פעם אחת (איך נראה כפתור ראשי, צבע שגיאה, גופן כותרת), הוא לא צריך ללמוד מחדש בכל מסך — מה שמפחית עומס קוגניטיבי. Variety הוא עיקרון נכון, אבל הוא מיועד להמנע מ*שעמום ויזואלי בתוך מסך* — לא לייצר אי-עקביות בין מסכים. שינויים אקראיים בין מסכים ישברו את תפיסת ה-Unity ויבלבלו משתמשים.
Learning Objective: Distinguish between the Repetition principle (cross-screen consistency) and Variety (preventing monotony within a screen).
Misconception: Students often think Variety means each screen should look different. In reality, Variety applies within a single layout to prevent monotony, while Repetition across screens is essential for a coherent experience.
