# Customization Rules

## Exam & Quiz Generation Rules
- **Avoid Length Bias in Correct Answers**: The longest answer option must not always be the correct one. Ensure correct options are sometimes shorter than, or equal in length to, incorrect options (distractors).
- **Distractor Quality & Dilemma Design**: Answer options must force users to read all choices. Design distractors to disqualify irrelevant options easily, but create a high-quality cognitive dilemma between 2 or 3 options that appear very similar. Ensure only one of these options is correct, with a subtle but definite reason (a specific detail, context mismatch, or technical inaccuracy) that disqualifies the others.
