# Quiz Checklist

Every generated quiz must satisfy:

- [ ] Questions progress from easy to hard.
- [ ] Multiple concepts are combined.
- [ ] Exactly one correct answer.
- [ ] Four plausible options.
- [ ] No joke distractors.
- [ ] Explanation for every question.
- [ ] Scenario questions use realistic organizations.
- [ ] Terminology is technically correct.
- [ ] Questions evaluate understanding.
- [ ] Output follows quiz template.
